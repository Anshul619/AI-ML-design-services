# The 5 Core Dimensions of RAG Evaluation

| Dimension              | Goal                                            |
| ---------------------- | ----------------------------------------------- |
| Retrieval Quality      | Did we retrieve the right context?              |
| Generation Quality     | Was the answer correct/helpful?                 |
| Grounding/Faithfulness | Did the answer stay within context?             |
| System Efficiency      | Was it fast/scalable/cost-efficient?            |
| Robustness             | Does it work consistently on difficult queries? |
