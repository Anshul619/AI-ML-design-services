# Embedding types

|                                      | Description                                   | Use Cases                                                                                      |
|--------------------------------------|-----------------------------------------------|------------------------------------------------------------------------------------------------|
| Dense Embeddings                     | Transforms text into dense semantic vectors.  | RAG<br/>- Semantic search<br/>- Chatbots<br/>- Recommendation systems                          |
| Sparse Embeddings                    | Keyword-oriented embeddings                   | exact keywords <br/>- enterprise <br/>- search ecommerce <br/>- log search <br/>- legal search |
| Hybrid Embeddings / Hybrid Retrieval | Combination of dense, sparse embeddings       | production retrieval                                                                           |
| Sentence Embeddings                  | Semantic meaning at sentence level            | short-text similarity                                                                          |
| Multimodal Embeddings                | Embeddings shared across text, images, videos | cross-media understanding                                                                      |
